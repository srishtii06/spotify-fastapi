# Spotify FastAPI Backend

A FastAPI-based backend application that integrates with the Spotify Web API. This project provides endpoints to:

- View the currently playing track
- Fetch top tracks
- Play a specific track
- Get followed artists
- Control playback

## Features

- ✅ OAuth 2.0 Authorization Code Flow
- ✅ Playback control (Play track)
- ✅ Top 10 tracks retrieval
- ✅ Followed artists list
- ✅ Now playing endpoint
- ✅ Clean and async FastAPI design
- ✅ Easy to deploy (Heroku/GitHub)

## Requirements

- Python 3.8+
- Spotify Developer Account with Client ID and Secret
- Spotify Premium account (required for playback controls)
